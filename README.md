# CT Neal-Herman - Personal Portfolio

Personal portfolio website for CT Neal-Herman, a Senior Software Engineer specializing in Ruby on Rails and React.

## About

This is a GitHub Pages site showcasing my professional experience and portfolio projects.

## Local Development

To run locally:

```bash
# Using Python (built into macOS)
python3 -m http.server 8000

# Then open http://localhost:8000
```

## Built With

- Bootstrap 3 (Bootswatch Flatly theme)
- Font Awesome
- jQuery

Based on the [Freelancer](http://startbootstrap.com/template-overviews/freelancer/) theme by [Start Bootstrap](http://startbootstrap.com/).
